# Fiscal Tax Flat Rate

Fixed tax amounts per fiscal region and validity period. Used together with base rates and euro norm adjustments in car tax calculations.

| Property      | Description                      |
| ------------- | -------------------------------- |
| Fiscal region | The region this rate applies to. |
| Start         | Start date of validity.          |
| Rate          | Flat tax amount.                 |
