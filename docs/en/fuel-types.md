# Fuel Types

Reference list of fuel types (petrol, diesel, electric, etc.), used in the simulation, car infos, and car price estimates.

| Property         | Description                                       |
| ---------------- | ------------------------------------------------- |
| Code             | Unique short identifier for the fuel type.        |
| Name             | Display name of the fuel type.                    |
| Active           | Whether the fuel type is available for selection. |
| Price per        | Price-per-unit value used in calculations.        |
| CO2 contribution | CO2 contribution value used in calculations.      |
