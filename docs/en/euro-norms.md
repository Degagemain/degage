# Euro Norms

Reference list of emission standards (e.g. Euro 6), with group and validity period. Used in car infos and car tax euro norm adjustments.

| Property | Description                                  |
| -------- | -------------------------------------------- |
| Code     | Unique short identifier for the norm.        |
| Name     | Display name of the euro norm.               |
| Group    | Group label used for tax adjustments.        |
| Active   | Whether the norm is available for selection. |
| Start    | Start date of validity.                      |
| End      | End date of validity (if applicable).        |

## Export

Use **More → Export** to download the current filtered/sorted list as CSV or JSON.
