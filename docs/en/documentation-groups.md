# Documentation groups

Use documentation groups to organise help articles into labelled buckets. You can assign each documentation entry to none or several groups.
Groups have a display order and a name in each supported language.

| Property | Description                                              |
| -------- | -------------------------------------------------------- |
| Order    | Sort position when listing groups (lower numbers first). |
| Name     | Shown in English, Dutch, and French.                     |

You can create, edit, import, and delete groups from the admin list. Deleting a group that is still linked to documentation may be blocked until
those links are removed.
