# Fiscal Tax Base

Base tax rates per fiscal region, by engine size (max cc, fiscal pk) and validity period. Used in car tax calculations (e.g. in the simulation
price step).

| Property      | Description                                    |
| ------------- | ---------------------------------------------- |
| Fiscal region | The region this rate applies to.               |
| Max cc        | Maximum engine capacity (cc) for this bracket. |
| Fiscal pk     | Fiscal power value.                            |
| Start         | Start date of validity.                        |
| End           | End date of validity (if applicable).          |
| Rate          | Tax rate value.                                |
