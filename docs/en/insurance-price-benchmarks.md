# Insurance

| Property      | Description                                       |
| ------------- | ------------------------------------------------- |
| Year          | Year this benchmark applies to.                   |
| Max car price | Maximum car price for this bracket (or no limit). |
| Base rate     | Base rate value.                                  |
| Rate          | Rate value.                                       |
