# Fiscale regio's

Referentielijst van regio's voor autobelastingregels. Belastingen basistarieven, vast tarief en euronormaanpassingen worden per fiscale regio
gedefinieerd. Er is momenteel slechts één fiscale regio: **Vlaanderen**. De berekening van de verkeersbelasting volgt de
[officiële Vlaamse regels (verkeersbelastingen personenwagens)](https://www.vlaanderen.be/belastingen-en-begroting/vlaamse-belastingen/verkeersbelastingen/verkeersbelastingen-voor-personenwagens).

| Eigenschap | Beschrijving                             |
| ---------- | ---------------------------------------- |
| Code       | Unieke korte identificatie van de regio. |
| Naam       | Weergavenaam van de fiscale regio.       |
| Standaard  | Of deze regio de standaard is.           |
