# Documentatiegroepen

Met documentatiegroepen orden je help-artikelen onder herkenbare labels. Elk document kan bij geen, één of meerdere groepen horen. Elke groep
heeft een weergavevolgorde en een naam in elke ondersteunde taal.

| Eigenschap | Beschrijving                                               |
| ---------- | ---------------------------------------------------------- |
| Volgorde   | Sorteerpositie in de groepenlijst (lagere getallen eerst). |
| Naam       | Getoond in het Engels, Nederlands en Frans.                |

Je kunt groepen aanmaken, bewerken, importeren en verwijderen vanuit de adminlijst. Verwijderen kan geweigerd worden zolang een groep nog aan
documentatie gekoppeld is.
