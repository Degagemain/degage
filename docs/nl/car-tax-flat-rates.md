# Vast tarief fiscale belasting

Vaste belastingbedragen per fiscale regio en geldigheidsperiode. Gebruikt samen met basistarief en euronormaanpassingen in de berekening van de
verkeersbelasting.

| Eigenschap    | Beschrijving                                  |
| ------------- | --------------------------------------------- |
| Fiscale regio | De regio waarop dit tarief van toepassing is. |
| Start         | Startdatum van de geldigheid.                 |
| Tarief        | Vast belastingbedrag.                         |
