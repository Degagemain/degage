# Gemeenten

Referentielijst van locaties (postcode, naam, gemeente), met optioneel provincie en hub. Gebruikt bij het uitvoeren van simulaties en voor
geografisch filteren.

| Eigenschap    | Beschrijving                                 |
| ------------- | -------------------------------------------- |
| Postcode      | Postcode.                                    |
| Naam          | Naam van de gemeente.                        |
| Gemeente      | Gemeente waartoe de locatie behoort.         |
| Provincie     | Provincie waartoe de gemeente behoort.       |
| Hub           | Hub waaraan de gemeente is toegewezen.       |
| Hoge vraag    | Of de gemeente is gemarkeerd als hoge vraag. |
| Actieve leden | Of de gemeente actieve leden heeft.          |
