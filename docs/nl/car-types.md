# Voertuigtypes

Referentielijst van voertuigtypes (bv. sedan, SUV), gekoppeld aan een merk en brandstoftype. Gebruikt in de simulatie, car infos en
prijsschatingen.

| Eigenschap | Beschrijving                              |
| ---------- | ----------------------------------------- |
| Merk       | Het automerk waar dit type bij hoort.     |
| Naam       | Weergavenaam van het voertuigtype.        |
| Brandstof  | Brandstoftype dat bij dit type hoort.     |
| Ecoscore   | Milieuscore (bv. A–E).                    |
| Actief     | Of het type beschikbaar is voor selectie. |

## Export

Gebruik **Meer → Exporteren** om de huidige gefilterde/gesorteerde lijst te downloaden als CSV of JSON.
