# Verzekering

Referentiegegevens verzekeringskosten per jaar: maximale autoprijs, basistarief en tarief. Gebruikt voor schattingen of rapportage.

| Eigenschap            | Beschrijving                                          |
| --------------------- | ----------------------------------------------------- |
| Jaar                  | Jaar waarop deze benchmark van toepassing is.         |
| Max autoprijs (excl.) | Maximale autoprijs voor deze schijf (of geen limiet). |
| Basistarief           | Waarde basistarief.                                   |
| €/km                  | Tariefwaarde.                                         |
