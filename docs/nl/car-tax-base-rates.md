# Belastingen basistarieven

Basistarief belastingen per fiscale regio, per cilinderinhoud (max cc, fiscal PK) en geldigheidsperiode. Gebruikt in de berekening van de
verkeersbelasting (bv. in de simulatiestap prijs).

| Eigenschap    | Beschrijving                                         |
| ------------- | ---------------------------------------------------- |
| Fiscale regio | De regio waarop dit tarief van toepassing is.        |
| Max cc        | Maximale cilinderinhoud (cc) voor deze schijf.       |
| Fiscal PK     | Fiscale pk-waarde.                                   |
| Start         | Startdatum van de geldigheid.                        |
| Einde         | Einddatum van de geldigheid (indien van toepassing). |
| Tarief        | Tariefwaarde.                                        |
