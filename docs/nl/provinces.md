# Provincies

Referentielijst van provincies, elk gekoppeld aan een fiscale regio. Gebruikt om gemeenten te groeperen en voor belasting-/regionale logica.

| Eigenschap    | Beschrijving                                    |
| ------------- | ----------------------------------------------- |
| Naam          | Weergavenaam van de provincie.                  |
| Fiscale regio | De fiscale regio waar deze provincie bij hoort. |
