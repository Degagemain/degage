# Fiscal Tax Flat Rate

Montants fixes par région fiscale et période de validité. Utilisés avec les taux de base et ajustements euro normes dans le calcul de la taxe
véhicule.

| Propriété      | Description                              |
| -------------- | ---------------------------------------- |
| Région fiscale | La région à laquelle ce taux s'applique. |
| Début          | Date de début de validité.               |
| Tarif          | Montant fixe.                            |

## Export

Utilisez **Plus → Exporter** pour télécharger la liste filtrée/triée en CSV ou JSON.
