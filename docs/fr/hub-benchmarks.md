# Hub Benchmarks

Données de référence par hub : km propriétaire, km partagés moy/min/max. Utilisées pour les rapports et la comparaison entre hubs.

| Propriété        | Description                            |
| ---------------- | -------------------------------------- |
| Hub              | Le hub auquel ce benchmark s'applique. |
| Km propriétaire  | Kilométrage propriétaire (km).         |
| Km partagés moy. | Kilométrage partagé moyen (km).        |
| Km partagés min. | Kilométrage partagé minimum (km).      |
| Km partagés max. | Kilométrage partagé maximum (km).      |

## Export

Utilisez **Plus → Exporter** pour télécharger la liste filtrée/triée en CSV ou JSON.

## Import

Utilisez **Plus → Importer** pour téléverser un fichier JSON obtenu précédemment via Exporter. Les enregistrements avec un id sont mis à jour ;
ceux sans id sont ajoutés. Chaque ligne affiche son propre statut, et un échec n'arrête pas les autres lignes.
