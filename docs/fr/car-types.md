# Types de véhicules

Liste de référence des types de véhicules (ex. berline, SUV), liés à une marque et un type de carburant. Utilisés dans la simulation, les car
infos et les estimations de prix.

| Propriété | Description                                  |
| --------- | -------------------------------------------- |
| Marque    | La marque à laquelle ce type appartient.     |
| Nom       | Nom d'affichage du type de véhicule.         |
| Carburant | Type de carburant associé à ce type.         |
| Écoscore  | Score environnemental (ex. A–E).             |
| Actif     | Si le type est disponible pour la sélection. |
