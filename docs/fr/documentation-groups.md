# Groupes de documentation

Les groupes de documentation servent à regrouper les articles d’aide sous des libellés. Chaque fiche documentation peut appartenir à aucun ou
plusieurs groupes. Chaque groupe a un ordre d’affichage et un nom dans chaque langue prise en charge.

| Propriété | Description                                                                         |
| --------- | ----------------------------------------------------------------------------------- |
| Ordre     | Position de tri dans la liste des groupes (les nombres les plus petits en premier). |
| Nom       | Affiché en anglais, néerlandais et français.                                        |

Vous pouvez créer, modifier, importer et supprimer les groupes depuis la liste d’administration. La suppression d’un groupe encore lié à de la
documentation peut être refusée tant que ces liens existent.
