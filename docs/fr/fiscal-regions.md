# Régions fiscales

Liste de référence des régions utilisées pour les règles de taxe automobile. Les taux de base, taux fixes et ajustements euro normes sont
définis par région fiscale. Il n'existe actuellement qu'une région fiscale : **Vlaanderen**. Le calcul de la taxe véhicule suit les
[règles officielles flamandes (verkeersbelastingen personenwagens)](https://www.vlaanderen.be/belastingen-en-begroting/vlaamse-belastingen/verkeersbelastingen/verkeersbelastingen-voor-personenwagens).

| Propriété | Description                               |
| --------- | ----------------------------------------- |
| Code      | Identifiant court unique de la région.    |
| Nom       | Nom d'affichage de la région fiscale.     |
| Défaut    | Si cette région est la région par défaut. |

## Export

Utilisez **Plus → Exporter** pour télécharger la liste filtrée/triée en CSV ou JSON.
