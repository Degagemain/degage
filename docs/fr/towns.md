# Communes

Liste de référence des localités (code postal, nom, commune), avec province et hub optionnels. Utilisée pour lancer les simulations et le
filtrage géographique.

| Propriété      | Description                                |
| -------------- | ------------------------------------------ |
| Code postal    | Code postal.                               |
| Nom            | Nom de la commune.                         |
| Commune        | Commune à laquelle la localité appartient. |
| Province       | Province à laquelle la commune appartient. |
| Hub            | Hub auquel la commune est assignée.        |
| Forte demande  | Si la commune est marquée forte demande.   |
| Membres actifs | Si la commune a des membres actifs.        |
