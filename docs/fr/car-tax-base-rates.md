# Fiscal Tax Base

Taux de base par région fiscale, par cylindrée (max cc, fiscal pk) et période de validité. Utilisés dans le calcul de la taxe véhicule (ex. à
l'étape prix de la simulation).

| Propriété      | Description                               |
| -------------- | ----------------------------------------- |
| Région fiscale | La région à laquelle ce taux s'applique.  |
| Max cc         | Cylindrée max. (cc) pour cette tranche.   |
| Fiscal pk      | Valeur de puissance fiscale.              |
| Début          | Date de début de validité.                |
| Fin            | Date de fin de validité (le cas échéant). |
| Tarif          | Valeur du taux.                           |
