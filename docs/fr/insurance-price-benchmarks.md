# Insurance

Données de référence des coûts d'assurance par année : prix véhicule max., taux de base et taux. Utilisées pour les estimations ou les rapports.

| Propriété    | Description                                             |
| ------------ | ------------------------------------------------------- |
| Année        | Année d'application de ce benchmark.                    |
| Prix max.    | Prix véhicule max. pour cette tranche (ou sans limite). |
| Taux de base | Valeur du taux de base.                                 |
| Taux         | Valeur du taux.                                         |

## Export

Utilisez **Plus → Exporter** pour télécharger la liste filtrée/triée en CSV ou JSON.
